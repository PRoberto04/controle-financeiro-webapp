# Guia de Configuração e Uso

Este é um guia para configurar e usar o projeto.

## Requisitos

- PHP >= 7.4
- Composer
- Node.js >= 14.x
- NPM ou Yarn
- Banco de Dados (ex: MySQL, SQLite)

## Configuração do Ambiente Local

1. **XAMPP**: Para configurar o ambiente local, você pode usar o XAMPP, um pacote de software livre contendo Apache, MySQL, PHP e Perl. [Baixe o XAMPP aqui](https://www.apachefriends.org/index.html).

## Clonando o Projeto

1. Clone o repositório para o seu ambiente local:

   ```bash
   git clone https://github.com/PRoberto04/controle-financeiro-webapp.git
   ```
## Configuração do Projeto

1. Instale as dependências do PHP executando o seguinte comando na raiz do projeto:

   ```bash
   composer install
   ```
2. Instale as dependências do Node.js executando o seguinte comando na raiz do projeto:

   ```bash
   npm install
   ```
## Iniciando o Servidor

Para iniciar o servidor PHP, execute o seguinte comando na raiz do projeto:

```bash
php artisan serve

```

## Executando as Migrações do Banco de Dados

Para executar as migrações do banco de dados e criar as tabelas necessárias, execute o seguinte comando na raiz do projeto:

```bash
php artisan migrate
```

## Compilando os Ativos JavaScript e CSS

Para compilar os ativos JavaScript e CSS, execute o seguinte comando na raiz do projeto:

```bash
npm run dev
```
